3. FAQ — Transparence sur les coûts des API

Ce que nous voulons que vous sachiez avant de signer.

Vais-je payer des abonnements après la mission ?

Non. Serafino Résout ne facture pas d'abonnements. Le prix de la mission est le prix total.

La Centralina utilise des API d'IA. Est-ce que ça me coûte quelque chose ?

Oui, mais très peu. Le système utilise des modèles d'IA pour traiter les demandes automatisées. Ces API sont facturées directement par le fournisseur sur votre carte — pas par Serafino Résout. Vous contrôlez le budget.

Usage mensuelCoût API estimé
Faible (50–100 requêtes/mois)~5–10 CHF
Moyen (200–500 requêtes/mois)~15–25 CHF
Intensif (1 000+ requêtes/mois)~40–80 CHF

Pour référence : si le système vous rend 4 h/semaine à 80 CHF/h, vous économisez ~1 280 CHF/mois. Les 15 CHF d'API représentent moins de 1,2 % de cette économie.

Que se passe-t-il si un fournisseur d'IA change ses prix ?

Le système est conçu avec des modèles de fallback — si un fournisseur devient trop cher, le système bascule automatiquement sur un autre. Votre budget API reste prévisible.

Je possède le code. Puis-je changer de fournisseur d'IA moi-même ?

Oui. Chaque mission inclut la propriété totale du code avec documentation. Vous pouvez modifier les fournisseurs ou faire intervenir n'importe quel développeur — sans dépendance vis-à-vis de Serafino Résout.
