# Centralina — Core product

## What it is

Centralina is the main operational system Serafino builds for service SMEs, craftspeople, and HORECA.
It is not a SaaS. It is not a template. It is built specifically for one business, deployed in 24 hours, and owned entirely by the client.

The simplest way to describe it: **a silent secretary that works 24/7 and never complains.**

She:
- Replies to quote requests automatically — in under 2 minutes
- Manages the team schedule — no conflicts, no calls
- Sends each operator their daily program every morning, directly on their phone
- Updates accounting (Bexio) automatically when the client signs a quote
- Gives the manager a real-time dashboard from their phone, 24/7

She does not take holidays. She does not ask for a raise. She does not cost more on weekends.

---

## How it works — concrete flow

### For the end client (the SME's customer):
1. Client sends a message (WhatsApp or Telegram)
2. Centralina collects the information needed for the quote
3. Quote PDF generated and sent automatically — under 2 minutes
4. Client signs → Bexio updated automatically

### For the team (operators):
- Every morning at a set time, each operator receives their day's schedule on Telegram
- No calls, no paper planning, no confusion
- Conflicts detected automatically before they happen

### For the manager:
- Real-time dashboard accessible from phone
- All quotes, planning, team status in one place
- No manual re-entry, no follow-up calls, no lost information

---

## What it is made of (without technical jargon)

- A system running on a secure server (always on, even when the manager is sleeping)
- Telegram bots: one for clients, one for operators, one for the manager
- A web dashboard the manager accesses from phone or computer
- Bexio integration (accounting software) — automatic
- Quote PDF generation — automatic

The client does not need to install anything. Their team uses Telegram (already on their phones). The manager uses a web interface.
